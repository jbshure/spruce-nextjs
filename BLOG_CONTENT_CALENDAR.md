## Content Creation Workflow

### Manual Process (Recommended with AI Assistance)
1. **Every 1st and 15th of month**: Create new blog post
2. Use Claude/ChatGPT to draft 2,500+ word article based on calendar topic
3. Review for accuracy and SEO optimization
4. Add to `/app/blog/[slug]/page.tsx`
5. Update `/app/blog/page.tsx` with new post in blogPosts array
6. Commit to Git and deploy

### Semi-Automated Process
1. Create drafts in advance (batch create 3-6 posts)
2. Schedule Git commits using GitHub Actions
3. Auto-deploy on commit via Vercel

### Fully Automated Process (Requires Development)
1. Set up OpenAI API integration
2. Create Node.js script to generate blog posts
3. GitHub Actions workflow to run on 1st and 15th
4. Auto-commit and deploy
5. **CRITICAL**: Include human review step before publishing

## SEO Best Practices

- **Word Count**: 2,000-3,000 words per post
- **Keywords**: 5-10 related keywords per post
- **Internal Links**: Link to 3-5 service pages per post
- **External Links**: 2-3 authoritative sources (CDC, FDA, OSHA)
- **Images**: 3-5 images per post with alt text
- **Meta Description**: 150-160 characters
- **URL Structure**: /blog/[keyword-rich-slug]
- **Related Posts**: Link to 3 related blog posts at bottom
